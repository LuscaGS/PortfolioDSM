VivaTShirt

Plataforma de e-commerce especializada no setor de vestuário, com gerenciamento de estoque, pedidos, clientes e integração com marketplaces.

Sobre o Projeto

A VivaTShirt é uma plataforma de e-commerce desenvolvida para oferecer uma solução completa de gerenciamento de vendas online no segmento de vestuário.

O sistema permite controlar estoque, pedidos e clientes de forma centralizada, além de possuir integração com a API da Shopee para sincronização de produtos e ampliação da presença em marketplaces.

A proposta da plataforma é unir a autonomia de uma loja virtual própria ao alcance de grandes canais de venda, proporcionando uma operação mais eficiente e moderna.

Tecnologias utilizadas: Node.js, HTML5, Tailwind CSS e MySQL.
