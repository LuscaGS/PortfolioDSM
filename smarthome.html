Smart Home

Plataforma de monitoramento residencial inteligente com backend em Kotlin, autenticação JWT e integração MQTT com dispositivos ESP32.

Sobre o Projeto

O Smart Home foi desenvolvido como Projeto Interdisciplinar com o objetivo de centralizar o gerenciamento de dispositivos inteligentes, sensores e automações residenciais em uma única plataforma.

A solução permite monitoramento em tempo real, gerenciamento de dispositivos, acompanhamento de sensores, criação de automações e recebimento de alertas quando eventos importantes são detectados.

O sistema é composto por uma API Backend desenvolvida em Kotlin com Spring Boot, uma aplicação Web para gerenciamento e um módulo IoT baseado em ESP32 conectado através do protocolo MQTT.

Tecnologias utilizadas: Kotlin, Spring Boot, Spring Security, JWT, PostgreSQL, MQTT, ESP32, JPA, Gradle e JUnit.

Kotlin Spring Boot JWT PostgreSQL MQTT ESP32 JPA JUnit

Principais Funcionalidades

• API REST com 14 CRUDs completos.

• Cadastro de usuários, casas, cômodos, sensores e dispositivos.

• Controle de automações, alertas e notificações.

• Histórico de leituras dos sensores.

• Autenticação e autorização utilizando JWT.

• Comunicação MQTT com dispositivos ESP32 em tempo real.

Minha Participação

Fui responsável pela implementação da integração IoT da solução, configurando a comunicação entre a API e os dispositivos ESP32 através de um broker MQTT.

Desenvolvi a autenticação utilizando JWT, garantindo segurança para acesso às rotas protegidas da aplicação.

Também participei da criação das DTOs (Data Transfer Objects), organizando o fluxo de dados entre controllers, services e clientes da API.

Além disso, trabalhei na integração dos comandos MQTT responsáveis pelo envio e recebimento de informações entre a plataforma e os dispositivos conectados.
